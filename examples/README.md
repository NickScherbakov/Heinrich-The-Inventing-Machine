# Heinrich Examples

This directory contains examples demonstrating how to use the Heinrich TRIZ Engine.

## Basic Usage

**File:** `basic_usage.py`

A simple example showing how to use the problem parser to analyze technical problems.

**Run it:**
```bash
python3 basic_usage.py
```

**What it demonstrates:**
- Creating a ProblemParser instance
- Parsing problem descriptions
- Extracting technical systems, improvements, and constraints
- Analyzing multiple problems
