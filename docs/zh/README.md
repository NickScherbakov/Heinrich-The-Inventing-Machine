## 关于Heinrich

Heinrich将经典的TRIZ（发明问题解决理论）方法论与现代大型语言模型相结合，提供系统化、可解释的发明问题解决能力。

以TRIZ创始人根里奇·阿奇舒勒命名，Heinrich体现了原始方法论的系统思维方式，同时利用当代人工智能的进步。

## 主要特性

- **系统化TRIZ流程**: 遵循经典TRIZ方法论的7模块工作流
- **可解释推理**: 每个步骤都被记录并可追溯到TRIZ原理
- **知识库**: 完整的39个参数、40个原理、矛盾矩阵
- **多语言**: 全面支持英语、中文、俄语和阿拉伯语
